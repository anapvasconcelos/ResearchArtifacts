# How to reproduce

This document provides step-by-step instructions on how to reproduce the results presented in the paper by generating all figures and tables reported in the study.

The artifact allows researchers to reproduce all figures and tables reported in the paper with a single command.

The general workflow of the artifact is:

**dataset → scripts → outputs**

The dataset serves as input for the scripts, which automatically generate the figures and tables reported in the paper.


The replication package is organized as follows:

```
replication-package/
├── processed-data/     - dataset and LaTeX files
├── scripts/            - R scripts used in the analysis
└── outputs/            - generated outputs
```

💡 The complete set of figures and tables can be reproduced automatically by executing the script `run-all-scripts.R` located in `replication-package/scripts/`.


## System requirements

No specific hardware requirements are necessary to execute the scripts.  

All analyses can be executed on a standard personal computer.


## Required software

To use the artifact, the following software is required:

### To run the scripts - `.R` files

- The **R programming language**
- An environment capable of running R scripts
  - e.g., RStudio or another compatible IDE.

### To open the script outputs - `.pdf` files

- Any **PDF reader**

### To inspect the dataset - `.xlsx` file

- Any **spreadsheet software** compatible with `.xlsx` files.
  - e.g., Excel, LibreOffice, Google Sheets.

### *Optional*: to compile figures and tables directly from LaTeX - `.tex` files

- A **LaTeX-compatible tool**

This step is optional and **not required to reproduce the results**, since all figures and tables can be generated automatically by the provided R scripts.

---

### Software versions used in the study

The artifact was tested using the following software versions:

- R version 4.4.3
- RStudio version 2024.12.1+563

Other recent versions of R and compatible environments should also work.

## Input Data

The dataset used in the analyses (`dataset.xlsx`) is located in: `replication-package/processed-data/`.

This file contains the cleaned and organized data collected from the analyzed papers, including derived variables, summary counts, and intermediate results used in the study.

The dataset serves as input for the reproduction scripts that generate the figures and tables reported in the paper.




## Running the scripts with R

### 1. Installing R

Download and install the R programming language:
- [R programming language](https://cran.r-project.org/)

---

### 2. Installing an R environment

Download and install an environment capable of running R scripts, such as RStudio.
- [RStudio](https://www.rstudio.com/products/rstudio/download/)

Other compatible environments may also be used, but for the purposes of this tutorial, we assume the use of RStudio.

---

### 3. Downloading the artifact

Download either:
- `replication-package.zip`

    **or**

- the complete artifact `research-artifact.zip`

---

### 4. Extracting the files

- Unzip the downloaded file.

- ***Important***: The folder structure must be preserved for the scripts to work correctly.

---


### 5. Executing scripts

To execute a script you need to:
- Open the script in RStudio.
- Run the script using the following shortcuts:
  - **Mac**: `Cmd` + `Shift` + `Enter`
  - **Windows/Linux**: `Ctrl` + `Shift` + `Enter`

Execution time: **less than 30 seconds**.

---

### 6. Execution modes

Two execution modes are available.



#### **6A. Run all scripts at once**

To generate **all figures and tables**:

1. Navigate to `replication-package/scripts`
2. Open `run-all-scripts.R`
3. Run the script



#### **6B. Run individual scripts**

If you want to **generate specific figures or tables**, or generate all of them individually:

1. Navigate to `replication-package/scripts/individual-scripts`
2. Run one of the following scripts:
   - `F02-FSE-timeline.R` → generates **Figure 2**
   - `F03-ICSE-timeline.R` → generates **Figure 3**
   - `F04-ICSE-FSE-timeline.R` → generates **Figure 4**
   - `F12-FSE-spearman.R` → generates **Figure 12**
   - `tex-figures-and-tables.R` → generates **Figures 5–11 and Tables 1–12**



---
